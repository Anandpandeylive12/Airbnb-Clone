# Airbnb Frontend

This is the frontend for an Airbnb-like application, built with React and Vite.

## Features

- Browse and search listings
- User authentication
- Book and manage stays
- Responsive UI

## Getting Started

1. Install dependencies:
   ```
   npm install
   ```
2. Start the development server:
   ```
   npm run dev
   ```

## Tech Stack

- React
- Vite
- Axios (for API calls)
